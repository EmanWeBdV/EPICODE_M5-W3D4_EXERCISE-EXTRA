## 🚀 Features

- **Country search**
  - Search countries through a text input
  - Perform API requests based on the entered country name
  - Display matching countries in a results list

- **Results rendering**
  - Show country data inside dedicated components
  - Pass data between components using **props**
  - Build a simple and reusable UI structure

- **Favorites management**
  - Save countries as favorites
  - Remove countries from favorites
  - Manage favorites globally using **React Context**

- **Favorites section**
  - Dedicated area to display saved countries
  - Easier state sharing across the application

- **Educational Context**
  - Built as a React exercise to practice API calls, props, Context API and state sharing across components
